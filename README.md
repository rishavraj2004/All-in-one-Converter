# 🌍 Unit Converter

A simple, fast, and responsive **Unit Converter** built using **HTML, CSS, and JavaScript**. This web application allows users to convert values across multiple measurement categories with an intuitive interface.

## 🚀 Features

- 📏 Length Converter
- ⚖️ Mass Converter
- 🌡️ Temperature Converter
- ⏱️ Time Converter
- 🚗 Speed Converter
- 🧪 Volume Converter
- 📐 Area Converter
- 📦 Data Converter
- 📱 Responsive Design
- ⚡ Instant Conversion
- 🎨 Clean and Modern UI

## 💻 Getting Started

### Clone the repository

```bash
git clone https://github.com/rishavraj2004/Unit-Converter.git
```

### Navigate to the project

```bash
cd Unit-Converter
```

### Open the project

Simply open `index.html` in your browser.

Or use VS Code Live Server for a better development experience.


## ✨ Supported Conversions

| Category | Examples |
|----------|----------|
| 📏 Length | Meter, Kilometer, Mile, Inch, Foot |
| ⚖️ Mass | Gram, Kilogram, Pound, Ounce |
| 🌡️ Temperature | Celsius, Fahrenheit, Kelvin |
| ⏱️ Time | Seconds, Minutes, Hours, Days |
| 🚗 Speed | km/h, m/s, mph |
| 🧪 Volume | Liter, Milliliter, Gallon |
| 📐 Area | Square Meter, Acre, Hectare |
| 💾 Data | Byte, KB, MB, GB, TB |
